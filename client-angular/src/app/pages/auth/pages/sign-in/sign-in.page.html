What is E-lab?

|||

E-lab is a community-driven grant program based on the passed Cyber Republic proposal.

|||

The main goal of E-Lab is to fast-track project development of dApps utilizing Elastos infrastructure through community grant instead of requesting funds from the Cyber Republic DAO.

Grant Process

  1. 1

    Download Elastos Essentials

    Download and install Elastos Essentials (EE) super wallet.

    iOS Android
  2. 2

    Create a DID (Decentralized Identifier)

    A DID (decentralized identity/identifier) will be automatically created during EE wallet onboarding. By issuing Decentralized Identities (DIDs) to everyone on the internet, a blockchain automates trust. In a blockchain-based system of exchange, transactions occur only between individuals, thereby removing the need for central authorities and intermediaries.

  3. 3

    Create and submit proposal on Cyber Republic

    You must first create a proposal on Cyber Republic (CR) website in order to create a proposal on E-lab. More info on how to create a proposal on CR here.

  4. 4

    Sign in to E-lab

    Sign in to E-lab by scanning the QR code from EE super wallet.

  5. 5

    Activate account

    Activate account either through KYC-me or Discord verification.

  6. 6

    Create community proposal

    Create a community proposal by linking the original proposal from the Cyber Republic website.

  7. 7

    Voting period

    7-day period community vote on active proposals. Developers are also encouraged to promote their proposals to community members using social media platforms.

  1. 8

    Start building!

    Grant distributed to winning proposal and developers can start building on Elastos!
